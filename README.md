## Test and Deploy

Use the built-in continuous integration in GitLab.

For test in this project we used jest and @testing-library/react-native.
running test

```
npm run test
```

Installing packages

```
npm install
```

or

```
yarn
```

Starting project on android emulator

```
npm run android
yarn run android
```

Starting project on ios emulator

```
npm run ios
yarn run ios
```
